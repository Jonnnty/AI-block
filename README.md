# AI-block

AI 相关工具集合（持续扩展中）。

## 模块

| 模块 | 说明 | 在线链接 |
|------|------|----------|
| **placement** | Gaussian Splat 建筑摆放 + 相机轨迹 | [打开摆放器](https://jonnnty.github.io/AI-block/placement/placement_editor.html) |

## 本地开发

各子目录见各自 README。摆放器：

```cmd
cd placement
pip install -r requirements.txt
python serve_editor.py
```

## GitHub Pages

推送 `main` 后 Actions 自动部署；首次需在仓库 **Settings → Pages → Source** 选 **GitHub Actions**。
